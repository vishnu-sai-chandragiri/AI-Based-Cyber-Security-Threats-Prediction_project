# Datasets

## Overview
This folder contains training datasets for ML models.

## Datasets Used

### 1. CICIDS2017
- **Size**: 2.8M network traffic records
- **Source**: Canadian Institute for Cybersecurity
- **Format**: CSV, PCAP
- **Labels**: Normal, DDoS, PortScan, BruteForce

### 2. Malware Detection Dataset
- **Size**: 500K samples
- **Source**: Kaggle
- **Format**: PE files
- **Labels**: Benign, Malware (8 types)

### 3. Phishing URLs
- **Size**: 100K URLs
- **Source**: PhishTank
- **Format**: Text file
- **Labels**: Legitimate, Phishing

## Download Instructions
Due to size constraints, datasets are not included in the repository.

Download from:
- CICIDS2017: https://www.unb.ca/cic/datasets/ids-2017.html
- Malware: https://www.kaggle.com/datasets/khaledelmadawy/malware-detection
- PhishTank: https://www.phishtank.com/

Place downloaded files in appropriate subfolders.
